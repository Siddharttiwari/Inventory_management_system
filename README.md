📦 Inventory & Order Management System

A production-ready full-stack system to manage products, customers, and orders in real time.

Built with modern technologies and fully containerized using Docker for easy deployment.


🌟 Overview

This system helps businesses:

Track inventory in real time
Manage customer data
Process and monitor orders
Automatically update stock levels

Fast, responsive, and fully API-driven


📦 Product Management
Add new products
Update pricing & stock
View product list
Delete products (with validation rules)

👥 Customer Management
Add customers
View customer list
Remove customers
Store basic contact details

🚚 Order Management
Create new orders
View all orders
Order details tracking
Automatic stock deduction
Automatic total price calculation


🛠️ Tech Stack

Frontend

React (Vite)

Backend

FastAPI (Python)

Database

PostgreSQL

DevOps

Docker
Docker Compose
